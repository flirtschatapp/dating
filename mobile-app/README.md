# FlirtsChat Mobile App (React Native / Expo starter)

This folder contains a clean starting structure for implementing the 2050 glassmorphism dating app UI.

## Suggested setup

```bash
npx create-expo-app@latest mobile-app
```

Then keep/merge this structure:

- `src/screens/LoginScreen.js`
- `src/screens/HomeFlirtsChatScreen.js`
- `src/api/client.js`

## API base URL

Set your API endpoint in `src/api/client.js`.
